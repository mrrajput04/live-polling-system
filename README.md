# Live Polling System

## Features
- Real-time vote updates
- Admin dashboard
- One vote per session
- Charts and analytics
- Socket.IO integration
- Docker support

## Tech Stack
- React
- Node.js
- Express
- Socket.IO

## Installation

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## Docker
docker-compose up --build
